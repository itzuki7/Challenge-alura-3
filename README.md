# Challenge-alura-3
Una api rest creada con spring boot para tener tu foro creado por ti mismo para que cualquier persona pueda tener el suyo 
teniendo un sistema completo de topicos persistencia de datos y seguridad avanzada 

#Tecnologías Utilizadas
Java 25 (OpenJDK)

Spring Boot 4.0.3

Spring Security (Autenticación & Autorización)

MySQL (Base de Datos)

Flyway (Control de versiones de base de datos)

JWT (JSON Web Token) (Auth0)

Maven (Gestión de dependencias)
